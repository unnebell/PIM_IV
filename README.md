# Curso superior de tecnologia Análise e Desenvolvimento de Sistemas - PIM IV

O objetivo desse é programa é aplicar os conhecimentos obtidos no 2º semestre sobre as disciplinas Linguagem e Técnicas de Programação, Engenharia de Software Ágil I e Programação Estruturada em C na prática, utilizando uma metodologia baseada em "banco de dados simples".

No qual é executado o arquivo PIMIV.exe e conforme o uso do programa é possível notar a criação automática de arquivos.txt no seu diretório.

São eles:
* Log de usuários
  - O arquivo Log_Usuarios.txt é um informativo que mostra a data que o usuário utilizou o programa.
  
* Lista de pacientes cadastrados
  - O arquivo Lista_Pacientes.txt é responsável por salvar os dados que foram cadastrados na utilização do programa, informando ao usuário todas informações do paciente como nome, idade, comorbidades e endereço.

* Lista de pacientes identificados como grupo de risco
  - Para facilidade ao usuário, o arquivo Pacientes_Risco.txt é gerado caso o paciente cadastrado apresente uma idade superior a 65 anos ou alguma comorbidade, tendo em vista que é apresentado o nome do paciente, sua idade, comorbidades identificadas e seu CEP.
